Chapter 1: Introduction to Machine Learning

Understanding the fundamentals of machine learning and its applications

What is Machine Learning?

Machine Learning is a subset of artificial intelligence that enables computers to learn and make decisions from data without being explicitly programmed for every task. It's like teaching a computer to recognize patterns and make predictions based on examples.

Machine learning algorithms build mathematical models based on training data to make predictions or decisions without being explicitly programmed to perform the task.

Types of Machine Learning

1. Supervised Learning: Learning with labeled examples (input-output pairs)

2. Unsupervised Learning: Finding patterns in data without labeled examples

3. Reinforcement Learning: Learning through interaction and feedback

# Simple example of supervised learning concept
# We have data (features) and labels (target)
features = [[1, 2], [2, 3], [3, 4], [4, 5]]  # Input data
labels = [3, 5, 7, 9]  # Expected outputs

# Goal: Learn the pattern to predict new outputs
# Pattern: output = (feature1 + feature2) * 1
print("Pattern discovered: sum of features equals label")

Key Concepts

Features: Individual measurable properties of observed phenomena

Training Data: Dataset used to teach the algorithm

Model: The mathematical representation learned from data

Prediction: Output generated by the model for new data

Chapter 3: Data Preprocessing

Cleaning and preparing data for machine learning algorithms

Why Data Preprocessing?

Real-world data is often incomplete, inconsistent, and contains errors. Data preprocessing is the crucial step of cleaning and transforming raw data into a format suitable for machine learning algorithms.

Data scientists spend approximately 80% of their time on data preprocessing. Quality data leads to better model performance.

Handling Missing Data

Missing data is a common problem that can significantly impact model performance if not handled properly.

# Handling missing data
import pandas as pd
import numpy as np

# Create sample data with missing values
data = {
    'Age': [25, 30, np.nan, 35, 28],
    'Salary': [50000, np.nan, 45000, 70000, 52000],
    'Experience': [2, 5, 3, np.nan, 4]
}
df = pd.DataFrame(data)

# Check for missing values
print("Missing values:")
print(df.isnull().sum())

# Fill missing values
df['Age'].fillna(df['Age'].mean(), inplace=True)
df['Salary'].fillna(df['Salary'].median(), inplace=True)
df.dropna(inplace=True)  # Drop rows with remaining NaN values

print("\nAfter handling missing values:")
print(df)

Feature Scaling

Different features often have different scales. Feature scaling ensures all features contribute equally to the model.

# Feature scaling example
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# Sample data

print(data_standard)

# Min-Max scaling (range 0-1)
scaler_minmax = MinMaxScaler()
data_minmax = scaler_minmax.fit_transform(data)
print("\nMin-Max scaled data:")
print(data_minmax)

Chapter 4: Linear Regression

Understanding and implementing linear regression from scratch

What is Linear Regression?

Linear regression is a fundamental machine learning algorithm that models the relationship between a dependent variable and independent variables by fitting a linear equation to observed data.

y = mx + b
where y = predicted value, m = slope, x = input feature, b = y-intercept

Linear Regression from Scratch

Let's implement linear regression using the mathematical approach with gradient descent.

# Linear Regression from scratch
import numpy as np
import matplotlib.pyplot as plt

class LinearRegression:
    def __init__(self, learning_rate=0.01, iterations=1000):
        self.learning_rate = learning_rate
        self.iterations = iterations
        
    def fit(self, X, y):
        # Initialize parameters
        self.m = 0  # slope
        self.b = 0  # intercept
        n = len(X)
        
        # Gradient descent
        for i in range(self.iterations):
            # Predictions
            y_pred = self.m * X + self.b
            
            # Calculate gradients
            dm = (-2/n) * sum(X * (y - y_pred))
            db = (-2/n) * sum(y - y_pred)
            
            # Update parameters
            self.m -= self.learning_rate * dm
            self.b -= self.learning_rate * db
    
    def predict(self, X):
        return self.m * X + self.b

# Example usage
X = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

model = LinearRegression()
model.fit(X, y)

predictions = model.predict(X)
print(f"Slope: {model.m:.2f}")
print(f"Intercept: {model.b:.2f}")
print(f"Predictions: {predictions}")

Using Scikit-learn

While understanding the math is important, scikit-learn provides optimized implementations for practical use.

# Linear Regression with scikit-learn
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

# Create and train model
model_sklearn = LinearRegression()
X_reshaped = X.reshape(-1, 1)  # sklearn expects 2D array
model_sklearn.fit(X_reshaped, y)

# Make predictions
y_pred_sklearn = model_sklearn.predict(X_reshaped)

# Evaluate model
mse = mean_squared_error(y, y_pred_sklearn)
r2 = r2_score(y, y_pred_sklearn)

print(f"Mean Squared Error: {mse:.2f}")
print(f"R¬≤ Score: {r2:.2f}")
print(f"Coefficient: {model_sklearn.coef_[0]:.2f}")
print(f"Intercept: {model_sklearn.intercept_:.2f}")

Sigmoid Function

The sigmoid function maps any real number to a value between 0 and 1, making it perfect for probability estimation.

# Sigmoid function implementation
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# Plot sigmoid function
z = np.linspace(-10, 10, 100)
y = sigmoid(z)

plt.figure(figsize=(8, 6))
plt.plot(z, y, 'b-', linewidth=2)
plt.title('Sigmoid Function')
plt.xlabel('z')
plt.ylabel('œÉ(z)')
plt.grid(True)
plt.axhline(y=0.5, color='r', linestyle='--', alpha=0.7)
plt.show()

print("Sigmoid of 0:", sigmoid(0))
print("Sigmoid of large positive number:", sigmoid(10))
print("Sigmoid of large negative number:", sigmoid(-10))
